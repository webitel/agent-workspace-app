# Як засетапити собі im-чат?

Актуально для 06.2026

>[!IMPORTANT]
> Наразі нові чати не розгорнуті на test env, тому працюємо з dev envʼом.

### 1. Створіть свою IM Chat Чергу

Admin -> Queues

### 2. Створіть свою флов схему чатів, яка розподілятиме чат на цю Чергу

`joinQueue` флов апплікейшен

### 3. Використовуючи HTTP Client (Postman, curl, etc.) створіть собі бота 

(POST `v1/bots`) [Swagger](https://swagger.webitel.com/#/webitel.im.api.gateway.v1.Bots/Bots_CreateBot), в `body` -> `schema_id` – передаєте id створеної вами флов схеми (попередній крок).

Далі цей бот буде доступний у списку контактів чатів

 (GET `v1/contacts`) [Swagger](https://swagger.webitel.com/#/webitel.im.api.gateway.v1.Contacts/Contacts_Search)

Увага! Це НЕ CRM контакти.

### 4. Пишете собі з користувача-"клієнта" на цього Бота, який через створену вище флов Схему розподілить чат на вашу Чергу

Я використовую `examples/vue` апплікейшен з [репо](https://github.com/webitel/chat-sdk-web) `chat-web-sdk`, де є демо чатілки. Вписуєте туди свій токен,
знаходите у списку контактів бота і пишете собі.

### 5. Ловите цей чат у Workspace App

`cli.allTask()` – вам прийде цей чат як таск, у якого є поле `task.distribute.communication.thread` – це і
є цей тред (його превʼю з обмеженою кількістю полів). Цього інфо має бути достатньо щоб намалювати
превʼю цього треда в UI, а вже на клік-відкриття цього треда, треба буде гетнути
повну інформацію про цього (користуємось `@webitel/chat-sdk-web`).
